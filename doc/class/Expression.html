Class: Expression

Defined in: coffeequate\src\Expression.coffee

Overview

Public interface for nodes. This is the main object that the user will deal with in Coffeequate. It wraps underlying nodes and terminals in a neat interface.

# (Expression) sub(substitutions, equivalencies = {}, substituteUncertainties = false, evaluateSymbolicConstants = false)

Substitute values into the Expression.

Parameters:

  • substitutions ( Object ) — A map of variable labels to their values. Values can be integers, Expressions, Terminals, or BasicNodes.
  • equivalencies ( Object ) — Optional. A map of variable labels to a list of equivalent variable labels.
  • substituteUncertainties ( Boolean ) — Optional. Whether to substitute values into uncertainties instead of variables (default false).
  • evaluateSymbolicConstants ( Boolean ) — Optional. Whether to evaluate symbolic constants (as opposed to leaving them symbolic) (default false).

Returns:

  • ( Expression ) — The Expression with substituted values.
