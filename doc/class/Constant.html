Defined in: coffeequate/src/terminals.coffee
Inherits: Terminal

Overview

A constant in the equation tree, e.g. 0.5 or 1/2. Can be represented as a rational (1/2) or a float (0.5). If a constant is produced by negative exponentiation it becomes a rational; if it is entered as an integer then it is a rational; if it is entered as a float, then it remains a float. Rational * Rational -> Rational Rational * Float -> Float Float * Float -> Float
