Constructor Details

# (void) constructor(numerator, denominator = 1, mode = "rational")

Constants in the equation tree, e.g. 0.5 or 1/2. Can be represented as a RATIONAL (1/2) or a FLOAT (0.5). If a constant is produced by a negative exponentiation (or division, but CQ doesn't use that) then it becomes a rational. If it is entered as an integer then it will also be a rational. If it is entered as a float, then it remains a float. Rational Rational -> Rational Rational Float -> Float Float * Float -> Float
