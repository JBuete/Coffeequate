# (Number) bindingStrength()

This tells us how strongly bound together the node is. As in, because x+y*z is parsed as x+(y*z), * binds more closely than + does. When we want to express (x+y)*z, we put the x+y Add node inside a Bracket node, which binds very tightly.

Returns:

  • ( Number ) — Binding strength.

# (DrawingNode) bracketIfNeeded(child)

Wraps the node in brackets if a node has lower precedence than this node.

Parameters:

  • child ( DrawingNode ) — The node to (potentially) bracket.

Returns:

  • ( DrawingNode ) — The bracketed child, or the original child.
